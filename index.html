Text-slider demo page

Text slider

Text-slider is an easy to use jquery plugin to make beautiful text sliders.

Easy to use

$('.slide').textSlider(); is all you need

Slide trough the night

Or stop at the last slide

Easy to use

Easy to use and costumize

$('.slide').textSlider(
	{
		timeout: 7500,
		slideTime: 500,
		height: 100
	}
);

timeout

The timeout setting determines how long it takes for the text-slider to change to the next slide

slideTime

The slideTime setting determines the duration of the fade-in / fade-out effects .

height

The height setting determines the top and bottom padding of the slides
